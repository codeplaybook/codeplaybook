Architecture Boundaries

Controllers, services, and repositories must respect their layer responsibilities. Controllers handle request and response mapping only. Services contain business logic and orchestrate repositories. Repositories encapsulate all data access.

Scope

TypeScript files under src/api/ in controllers, services, and repositories subdirectories.

Rules

  • Controllers must not import repositories directly — delegate work to services.
  • Services must not call data-access methods directly — delegate to repositories.
  • Repositories must contain only data-access logic — no business rules or validation.
  • Cross-module dependencies go through public module exports, not direct file imports.
